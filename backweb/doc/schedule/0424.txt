자바 프로그래밍에서 객체(배열, 문자열, 클래스)

### 1. 흥미로운 도입
우리는 매일 수많은 데이터와 정보를 처리합니다. 스마트폰 앱, 웹사이트, 게임 등 
모든 소프트웨어 제품은 수많은 데이터를 저장하고 관리하는 방법이 필요합니다. 
이러한 데이터를 효율적으로 처리하고 관리하는 방법 중 하나가 바로 
'객체 지향 프로그래밍'입니다. 자바는 객체 지향 프로그래밍 언어로, 
객체를 사용하여 데이터와 행동을 캡슐화
(음성정보-귀를 통해서만 저장, 본인이 판단해서 입을 통해서만 출력되게 처리)합니다. 
오늘 우리는 자바의 기본적인 객체 구조인 배열, 문자열, 클래스를 배워 볼 것입니다.

### 2. 학습 목표
- 배열, 문자열, 클래스가 무엇인지 이해합니다.
- 자바에서 배열과 문자열을 어떻게 사용하는지 학습합니다.
- 클래스의 기본 구조와 객체 생성 방법을 익힙니다.
- 실제 예제를 통해 이론을 실습으로 연결합니다.

### 3. 학습 내용 계열화에 따른 단계별 순서
1. **배열 이해하기**: 배열의 정의, 생성, 초기화
2. **문자열 다루기**: 문자열 클래스 이해, 문자열 함수 사용법
3. **클래스와 객체**: 클래스 정의, 필드와 메소드, 생성자
4. **객체의 상호작용**: 객체 간의 데이터 교환과 메소드 호출
5. **상속과 다형성**: 기본 클래스와 파생 클래스, 오버라이딩과 오버로딩

### 4. 주요 핵심 키워드와 핵심개념
- **키워드**: 배열, 문자열, 클래스, 인스턴스, 메소드, 상속, 다형성
- **핵심 개념**:
  - 배열: 연속적인 메모리 공간에 동일한 타입의 데이터를 저장하는 자료구조
  - 문자열: 문자의 집합을 쉽게 다루기 위한 클래스
  - 클래스: 객체의 청사진, 데이터와 행동의 캡슐화
  - 상속: 코드 재사용성 향상, 유지보수 용이성 제공

### 5. 실무적용사례
온라인 쇼핑몰의 주문 시스템에서는 다양한 상품 정보를 배열로 관리하고, 고객 정보를 객체로 취급하여 주문 처리 과정에서 이들 데이터를 활용합니다. 주문 클래스는 고객 객체와 상품 객체를 연결하고, 주문 내역을 생성하여 데이터베이스에 저장하는 메소드를 포함합니다.

### 6. 퀴즈와 토론내용
**퀴즈 예시**:
1. 자바에서 배열을 선언하고 초기화하는 방법은 무엇인가요?
2. 문자열의 immutability(불변성)는 무엇을 의미하며 왜 중요한가요?
3. 클래스와 인스턴스의 차이점을 설명하세요.
