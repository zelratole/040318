** JSP(Java Server Pages)의 내장 객체

### 1. 흥미로운 도입
- **예시**: 웹 페이지 방문 시 맞춤형 환영 메시지가 표시된 경험에 대해 생각해 보세요. 이는 바로 JSP 내장 객체의 실제 사용 사례 중 하나입니다. 이 장에서는 JSP가 어떻게 웹 서버와 상호작용하여 동적인 웹 페이지를 생성하는지를 알아보겠습니다.

### 2. 학습목표
- **예시**:
  - JSP의 기본 개념과 내장 객체의 역할을 이해합니다.
  - 각 내장 객체의 기능과 사용 방법을 배웁니다.
  - 실제 웹 애플리케이션에서 JSP 내장 객체를 활용하는 방법을 학습합니다.

### 3. 학습내용계열화에 따른 단계별 순서
- **예시**:
  - **단계 1**: JSP 기초와 내장 객체 소개
  - **단계 2**: 각 내장 객체의 세부 기능 설명
  - **단계 3**: 객체 별 사용 사례와 코드 예시
  - **단계 4**: 내장 객체를 사용한 복잡한 동적 웹 페이지 구현

### 4. 주요 핵심 키워드와 핵심개념
- **핵심 키워드**: `request`, `response`, `session`, `application`, `out`, `pageContext`, `config`, `exception`
- **핵심 개념 예시**:
  - **request**: 클라이언트로부터 온 요청 정보를 담는 객체
  - **response**: 서버가 클라이언트로 보내는 응답을 관리하는 객체
  - **session**: 사용자의 세션 정보를 관리하는 객체
  - **application**: 애플리케이션의 전역 정보에 접근할 수 있는 객체

### 5. 실무적용사례
- **예시**:
  - **사례 1**: 온라인 쇼핑몰에서 사용자의 로그인 정보를 `session` 객체를 사용해 관리하는 방법
  - **사례 2**: 사용자가 제출한 폼 데이터를 `request` 객체를 통해 받아 처리하는 방법
  - **사례 3**: `application` 객체를 활용하여 애플리케이션의 방문자 수를 기록하는 방법
  - **사례 4**: `response` 객체를 사용하여 특정 파일을 사용자에게 다운로드하도록 제공하는 방법
