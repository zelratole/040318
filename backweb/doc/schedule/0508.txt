**  오라클 데이터베이스 SQL
### 1. 흥미로운 도입
오라클 데이터베이스는 전 세계 기업들이 대량의 데이터를 효율적으로 관리하고 분석하기 위해 사용하는 
강력한 관계형 데이터베이스 관리 시스템입니다. 이를 학습함으로써 여러분은 데이터의 추출, 저장, 수정 및 
검색 방법을 배울 뿐만 아니라, 실제 비즈니스 환경에서 데이터를 어떻게 활용하는지에 대한 깊은 이해를 얻을 수 있습니다.

### 2. 학습목표
- 기본적인 SQL 문법과 구조 이해하기
- 데이터 조회, 추가, 수정, 삭제 등의 기본적인 데이터 조작 기술 습득
- 복잡한 쿼리 작성을 통한 데이터 분석 능력 향상
- 데이터베이스 설계 및 정규화 개념 이해

### 3. 학습내용 계열화에 따른 단계별 순서
1. **SQL 기초**: SELECT, INSERT, UPDATE, DELETE 등 기본적인 SQL 명령어 학습
2. **조인과 서브쿼리**: 여러 테이블을 조합하고, 복잡한 데이터 집합을 추출하는 방법 학습
3. **데이터베이스 설계**: 테이블 생성, 스키마 설계 및 데이터 무결성 유지 방법 학습
4. **고급 SQL 기능**: 인덱싱, 스토어드 프로시저, 트리거 등 고급 기능에 대한 이해
5. **성능 최적화**: 쿼리 성능 분석 및 최적화 기술 습득

### 4. 주요 핵심 키워드와 핵심개념
- **키워드**: SELECT, JOIN, TRANSACTION, NORMALIZATION, INDEX
- **핵심개념**:
  - **조인**: 두 개 이상의 테이블을 연결하여 데이터를 검색하는 방법
  - **정규화**: 중복을 최소화하고 데이터 무결성을 향상시키기 위한 데이터베이스 설계 기법
  - **인덱스**: 데이터 검색 속도를 향상시키기 위한 데이터 구조
  - **트랜잭션**: 데이터의 정확성을 보장하기 위한 작업 단위

### 5. 실무적용사례
- **예제 1**: 온라인 쇼핑몰의 주문 관리 시스템에서 데이터베이스를 사용하여 고객 주문을 추적하고 관리하는 방법.
- **예제 2**: 금융 회사에서 대규모 금융 거래 데이터를 분석하여 사기 거래를 식별하는 데 사용하는 고급 SQL 쿼리.
- **예제 3**: 병원 정보 시스템에서 환자의 의료 기록을 관리하고 의사가 필요한 정보를 신속하게 조회할 수 있도록 
          지원하는 데이터베이스의 구축.
