# 데이터의 산술 연산자와 문자열 연결 처리, 그리고 WHERE 문을 활용하는 방법
### 1. 흥미로운 도입
상상해보세요. 당신은 대규모 온라인 쇼핑몰에서 데이터 분석가로 일하고 있습니다. 
갑자기, 마케팅 팀에서 이번 달 가장 많이 팔린 제품의 평균 가격과 해당 제품들의 
총 판매액을 급하게 요청했습니다. 여러분의 도전은 이 모든 정보를 단 몇 분 내에 찾아내는 것입니다. 
이제 SQL의 힘을 사용하여 이 문제를 어떻게 해결할 수 있는지 배워보겠습니다.

### 2. 학습목표
- 오라클 SQL에서 산술 연산자를 사용하여 데이터를 조작하는 방법을 이해하고 조작합니다.
- 문자열 연결 연산자를 사용하여 정보를 형식화하는 방법을 배우고 정확하게 구현할 줄 압니다.
- WHERE 문을 사용하여 특정 조건에 맞는 데이터만 검색하는 방법을 마스터합니다.

### 3. 주요 핵심 키워드와 핵심개념
**키워드:**
- 산술 연산자: +, -, *, /
- 문자열 연결 연산자: `||`  : '데이터베이스 문자열데이터'||'문자열'||45   
- WHERE 문
- select 컬럼단위 선택, 
  from 테이블 
  where 비교연산식/논리식 : 행단위 filtering

**핵심 개념:**
- **산술 연산자**: 데이터에 대한 기본적인 수학적 연산을 수행하여 새로운 결과를 생성합니다.
- **문자열 연결 연산자**: 여러 개의 문자열을 하나로 결합하여 보다 의미 있는 정보를 형성합니다.
- **WHERE 문**: 데이터베이스에서 특정 조건을 만족하는 행만을 선택하는데 사용됩니다.

### 4. 퀴즈 문제/토론 과제
**퀴즈 문제:**
1. 산술 연산자를 사용하여 '가격'과 '수량' 필드를 가진 테이블에서 각 제품의 총 판매 금액을 계산하는 SQL 쿼리는 어떻게 작성할까요?
2. '이름'과 '성' 필드를 결합하여 전체 이름을 생성하는 SQL 쿼리는 무엇일까요?

**토론 과제:**
- WHERE 문을 사용하는 것이 데이터베이스 성능에 어떤 영향을 미치는지 토론해봅시다.

### 5. 실무적용 사례
대형 병원에서 환자의 의료 기록을 관리하는 시스템을 구축하는 데이터베이스 관리자는 다음과 같은 요구 사항을 처리해야 합니다:
- 모든 환자의 나이와 성별을 사용하여 특정 연령대(예: 30-40세)의 여성 환자 목록을 작성합니다.
- 각 환자의 이름과 성을 결합하여 전체 이름을 만든 후, 특정 질병(예: 당뇨병)을 가진 환자를 찾아내야 합니다.
