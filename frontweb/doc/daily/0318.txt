### 웹프로그래밍 HTML 시작

#### 1. 흥미로운 도입

웹의 세계로 첫 발을 내딛는 여러분을 환영합니다! 여러분이 매일 방문하는 멋진 웹사이트들, 그 모든 것이 바로 HTML로 시작됩니다. 마치 거대한 건축물을 지탱하는 기초 공사와 같죠. HTML 없이는 웹 페이지의 기본 구조를 만들 수 없으며, 디지털 세계에서 우리의 아이디어를 현실로 만드는 첫 걸음이 됩니다. 준비되셨나요? 이제부터 여러분도 웹의 창조자가 될 수 있습니다!

#### 2. 학습목표

- **HTML 기본 이해**: HTML이 무엇인지, 왜 중요한지에 대한 기본적인 이해를 합니다.
- **기본 태그 익히기**: 제목, 단락, 링크, 이미지와 같은 기본적인 HTML 태그의 사용법을 배웁니다.
- **웹 페이지 구조 파악하기**: HTML 문서의 기본 구조를 이해하고, 자신만의 간단한 웹 페이지를 만들 수 있습니다.
- **실무적 적용**: 학습한 HTML 지식을 실제 프로젝트에 적용해 볼 수 있는 기회를 가집니다.

#### 3. 간단한 퀴즈

1. HTML은 무엇의 약자인가요?
   - A) Hyper Text Markup Language
   - B) High Tech Managing Language
   - C) Hyperlink and Text Markup Language
   - D) None of the above

2. 웹 페이지의 제목을 나타내는 태그는 무엇인가요?
   - A) `<title>`
   - B) `<header>`
   - C) `<head>`
   - D) `<h1>`

3. 이미지를 웹 페이지에 삽입하기 위해 사용하는 태그는 무엇인가요?
   - A) `<img>`
   - B) `<picture>`
   - C) `<image>`
   - D) `<src>`

#### 정답
1. A) Hyper Text Markup Language
2. A) `<title>`
3. A) `<img>`

#### 4. 주요 핵심 키워드와 개념

- **태그(Tag)**: HTML 요소를 생성하는데 사용되는 꺾쇠 괄호로 둘러싸인 코드입니다.
- **요소(Element)**: 시작 태그와 종료 태그, 그리고 그 사이의 내용을 포함하는 전체 구조를 말합니다.
- **속성(Attribute)**: 요소의 추가 정보를 제공하는 이름과 값의 쌍입니다. 예를 들어, 이미지 태그(`<img>`)에서 `src`는 이미지의 소스 URL을 나타냅니다.
- **구조적 요소**: 웹 페이지의 구조를 정의하는 태그로, `<header>`, `<footer>`, `<nav>`, `<section>` 등이 있습니다.
- **인라인 요소 vs 블록 요소**: 인라인 요소는 다른 요소와 같은 줄에 나타나는 반면, 블록 요소는 새로운 줄에서 시작합니다.

#### 5. 실무적용

이제 기본적인 HTML 지식을 습득했으니, 간단한 개인 프로젝트를 시작해 보세요. 예를 들어, 자기소개 페이지를 만들어보는 것입니다. 다음 요소들을 포함시키세요:

- 제목과 부제목
- 본문에는 여러 단락을 사용해 자기소개
- 취미나 관심사를 나타내는 이미지
- 이메일로 연락할 수 있는 링크
