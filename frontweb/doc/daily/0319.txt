블로그 글이나 웹 페이지 콘텐츠를 작성할 때, HTML을 이해하는 것은 필수적입니다. 
HTML은 웹 페이지를 구성하는 데 사용되는 표준 마크업 언어로, 다양한 요소를 통해 
콘텐츠를 구성하고 스타일을 적용합니다.
 이번 글에서는 HTML의 기본 요소 중 하나인 블록 태그와 인라인 태그, 이미지 삽입, 
 리스트 만들기, 테이블 만들기에 대해 알아보겠습니다. 

### 흥미로운 도입

웹 페이지는 마치 레고 블록처럼 다양한 구성 요소들로 이루어져 있습니다. 
각 블록은 웹 페이지의 다른 부분과 어우러져 정보를 전달하고 사용자 경험을 향상시킵니다. 
블록 태그와 인라인 태그는 이 블록들을 어떻게 배치할지, 이미지와 리스트, 테이블은 
웹 페이지에 어떤 정보를 어떻게 표현할지 결정하는 데 도움을 줍니다. 

### 학습목표

- 블록 태그와 인라인 태그의 차이점을 이해합니다.
- 이미지를 웹 페이지에 삽입하는 방법을 배웁니다.
- 순서가 있는 리스트와 순서가 없는 리스트를 만드는 방법을 알아봅니다.
- HTML을 사용하여 테이블을 만드는 기법을 익힙니다.

### 간단한 퀴즈

1. `<p>` 태그는 블록 태그와 인라인 태그 중 어디에 속하나요?
2. 이미지를 삽입할 때 사용하는 태그는 무엇인가요?
3. 순서가 있는 리스트를 만들 때 사용하는 태그는 무엇인가요?

### 주요 핵심 키워드와 개념

- **블록 태그(Block Tags)**: 새로운 줄에서 시작하며, 
	주로 구조를 정의하는데 사용됩니다. 
	예: `<div>`, `<p>`, `<h1>` 등.
- **인라인 태그(Inline Tags)**: 같은 줄에 나란히 배치되며, 
	텍스트나 이미지 같은 콘텐츠를 감싸는데 사용됩니다. 
	예: `<span>`, `<img>`, `<a>` 등.
- **이미지 삽입(Inserting Images)**: `<img>` 태그를 
	사용하여 이미지를 웹 페이지에 삽입합니다. `src` 속성으로 이미지 경로를 지정합니다.
- **리스트 만들기(Creating Lists)**: 순서가 있는 리스트는 
	`<ol>` 태그를, 순서가 없는 리스트는 `<ul>`. 각 항목은 `<li>` 태그로 표현됩니다.
- **테이블 만들기(Creating Tables)**: `<table>` 태그를 
	사용하여 테이블을 만들고, `<tr>`, `<th>`, `<td>` 태그로 행과 열, 셀을 정의합니다.

### 실무적용
웹 개발자뿐만 아니라, 콘텐츠 크리에이터, 마케터, 교육자 등 다양한 분야의 
	전문가들이 HTML을 활용하여 자신의 아이디어와 정보를 효과적으로 전달할 수 있습니다.
- **블로그 작성**: 인라인 태그와 블록 태그를 적절히 사용하여 글의 구조를 명확하게 하고,
	 이미지와 리스트를 활용하여 내용을 보다 풍부하게 표현할 수 있습니다.
- **웹사이트 구성**: 테이블을 사용하여 데이터를 구조화하고, 사용자가 정보를 쉽게 
	이해하고 접근할 수 있도록 도와줍니다.
- **교육 자료 제작**: 순서가 있는 리스트와 순서가 없는 리스트를 활용하여 학습 내용을 
	체계적으로 나열하고, 중요한 개념이나 정의를 강조할 때 인라인 태그를 사용할 수 있습니다.
