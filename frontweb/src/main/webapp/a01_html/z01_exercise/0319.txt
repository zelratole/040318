# 파일명을 자유롭게 생성
### 문제 1: 개인 정보 표 만들기

**요구사항:**

- `table`을 사용하여 개인 정보를 담은 표를 만드세요.
- 표에는 이름, 나이, 이메일 등의 열이 포함되어야 합니다.
- `thead`, `tbody`, `tfoot`을 사용하여 표를 구성하세요.
- `tfoot`에는 간단한 메모나 주의사항을 넣어주세요.

### 문제 2: 제품 목록 페이지 만들기

**요구사항:**

- `ul`과 `li` 태그를 사용하여 제품 목록을 만드세요.
- 각 제품명 옆에는 `span` 태그를 사용하여 가격을 표시하세요.

### 문제 3: 블로그 포스트 작성

**요구사항:**

- 제목은 `h1` 태그로, 부제목은 `h2` 태그로 작성하세요.
- 본문 내용은 `p` 태그를 사용하세요. 중요한 키워드는 `strong` 태그로 강조하세요.
- 본문에 이미지를 삽입하세요(`img` 태그 사용, `alt` 속성 포함).

### 문제 4: 연락처 목록 만들기

**요구사항:**

- `dl`, `dt`, `dd` 태그를 사용하여 연락처 목록을 만드세요.
- 각 연락처 항목에는 이름, 전화번호, 이메일 정보가 포함되어야 합니다.
- `dl` 태그를 사용하여 각 정보의 제목(`dt`)과 내용(`dd`)을 구분하세요.

### 문제 5: 사진 갤러리 페이지 만들기

**요구사항:**

- `div` 태그를 사용하여 각 사진을 감싸는 컨테이너를 만드세요.
- 각 사진(`img` 태그 사용) 아래에는 `figcaption` 태그를 사용하여 사진 설명을 추가하세요.
- 사진은 최소 3개 이상 포함시키세요.
- 각 사진에는 다른 `width`와 `height` 속성 값을 적용해 다양한 크기를 표현해 보세요.
