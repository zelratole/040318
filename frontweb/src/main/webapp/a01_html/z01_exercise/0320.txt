### 문제 1: 연락처 목록과 세부 정보 페이지 만들기

**요구사항:**
- `index.html` 페이지에는 각 연락처의 이름이 하이퍼링크로 나열되어 있어야 합니다.
- 하이퍼링크는 각각의 연락처에 대한 세부 정보를 담고 있는 별도의 HTML 페이지(예: `contact1.html`, `contact2.html`)로 연결되어야 합니다.
- 각 세부 정보 페이지에는 연락처의 이름, 전화번호, 이메일 주소가 표 형식으로 표시되어야 합니다.
- 세부 정보 페이지에서 `index.html` 페이지로 돌아갈 수 있는 하이퍼링크를 제공해야 합니다.

### 문제 2: 제품 소개 페이지 만들기

**요구사항:**
- 제품 소개 페이지에서는 다양한 제품들이 그리드 형태로 표시되어야 합니다.
- 각 제품은 제품 이미지(`img` 태그 사용)와 제품 이름을 포함해야 합니다.
- 제품 이미지는 클릭 시 제품에 대한 상세 설명을 담고 있는 별도의 HTML 페이지로 이동해야 합니다.
- 상세 설명 페이지에서는 제품 이미지, 설명, 가격 정보가 표시되어야 합니다.

### 문제 3: FAQ 섹션 만들기

**요구사항:**
- FAQ(자주 묻는 질문) 섹션을 포함하는 페이지를 만드세요.
- 각 질문은 `h3` 태그를 사용하여 표시하고, 답변은 바로 아래에 `p` 태그를 사용하여 제공합니다.
- 페이지 상단에는 각 질문으로 바로 이동할 수 있는 목차를 하이퍼링크 리스트로 제공하세요.

### 문제 4: 인라인 프레임을 사용하여 외부 컨텐츠 포함하기

**요구사항:**
- 인라인 프레임(`iframe`)을 사용하여 외부 웹사이트(예: 공식 문서, 튜토리얼 페이지)를 포함하는 페이지를 만드세요.
- 페이지 상단에는 몇 가지 외부 리소스로 바로 이동할 수 있는 하이퍼링크를 포함해야 합니다.
- 사용자가 하이퍼링크를 클릭할 때, 해당 컨텐츠가 현재 페이지 내의 `iframe`에서 로드되어야 합니다.

### 문제 5: 이벤트 페이지 만들기

**요구사항:**
- 이벤트 페이지에는 다가오는 이벤트 목록이 표시되어야 합니다.
- 각 이벤트 항목에는 이벤트 이름, 날짜, 장소가 포함되어야 합니다.
- 이벤트 이름은 상세 정보를 담고 있는 별도의 페이지로의 링크가 되어야 합니다.
- 상세 페이지에서는 이벤트에 대한 자세한 설명, 시간, 참여 방법 등이 제공되어야 합니다.
- 페이지 하단에는 `iframe`을 사용하여 이벤트 장소의 지도를 표시하세요.
