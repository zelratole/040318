### Vue와 스프링을 연동한 CRUD

#### 1. 흥미로운 도입

- **문제 제기**: 웹 애플리케이션을 개발하는 과정에서 프론트엔드와 백엔드의 상호작용이 중요함을 설명합니다. 예를 들어, 간단한 To-Do 리스트 애플리케이션이 있을 때, 데이터를 저장하고 불러오는 과정이 어떻게 이루어지는지 질문합니다.
- **실제 사례 소개**: 유명한 웹 애플리케이션, 예를 들어 트렐로나 구글 킵과 같은 서비스가 어떻게 프론트엔드와 백엔드가 연동되어 있는지 간략히 소개합니다.

#### 2. 학습목표

- Vue와 스프링을 연동하여 CRUD(Create, Read, Update, Delete) 기능을 구현할 수 있다.
- 프론트엔드(Vue)와 백엔드(Spring) 간의 통신을 이해하고 적용할 수 있다.
- RESTful API의 기본 개념과 사용 방법을 이해한다.
- 실무에서 자주 사용되는 기술 스택을 익히고, 이를 프로젝트에 적용할 수 있다.

#### 3. 주요 핵심 키워드와 핵심개념

- **Vue.js**: 프론트엔드 프레임워크로서의 역할과 기본 개념 (컴포넌트, 데이터 바인딩, 이벤트 처리 등)
- **Spring Boot**: 백엔드 프레임워크로서의 역할과 기본 개념 (의존성 주입, RESTful 서비스, 데이터베이스 연동 등)
- **CRUD**: 각 기능의 정의와 역할 (Create, Read, Update, Delete)
- **RESTful API**: REST의 기본 원칙과 HTTP 메서드 (GET, POST, PUT, DELETE)
- **Axios**: Vue에서 HTTP 요청을 보내기 위한 라이브러리

#### 4. 향상된 암기 기법에 의한 핵심 키워드와 핵심 개념 암기

- **키워드 연결법**: Vue의 'V'와 Spring의 'S'를 시각적으로 연결하여 "VS"처럼 기억. 두 프레임워크가 대결하는 것이 아닌 협력하는 관계로 이미지화.
- **스토리텔링**: 사용자가 Todo 애플리케이션을 사용하는 시나리오를 만들어 각 CRUD 작업이 사용자의 행동과 어떻게 연결되는지 이야기로 풀어 설명.
- **마인드맵**: Vue와 Spring을 중심으로 CRUD 기능이 어떻게 연결되는지 시각적으로 표현한 마인드맵을 제작.

#### 5. 실무적용사례

- **회사 프로젝트**: 특정 회사에서 Vue와 Spring을 사용해 프로젝트를 진행한 사례 소개. 예를 들어, 전자상거래 웹사이트에서 상품 관리 시스템을 Vue와 Spring으로 구축한 사례.
- **오픈 소스 프로젝트**: GitHub에서 Vue와 Spring을 사용한 오픈 소스 프로젝트 예시를 보여주고, 이를 분석해보는 시간.
- **개인 프로젝트**: 학생들이 직접 Vue와 Spring을 연동하여 작은 프로젝트(예: 간단한 블로그 시스템)를 만들어보게 합니다. 프로젝트 계획부터 구현, 테스트까지 전 과정을 경험하게 하여 실무 적용 능력을 기릅니다.
